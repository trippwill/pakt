---

## CLI

```sh
go install github.com/trippwill/pakt@latest
```

Requires Go 1.23 or later.

## Library

```go
import "github.com/trippwill/pakt/encoding"
```

Add to your module:

```sh
go get github.com/trippwill/pakt/encoding
```
