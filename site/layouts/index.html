A typed data interchange format where every value carries its type.

greeting:str     = 'hello world'
count:int        = 42
active:bool      = true
server:{host:str, port:int} = { 'localhost', 8080 }
Read the Docs GitHub

Every value has a type

No inference, no ambiguity. Type annotations are producer assertions checked at parse time. Documents are self-validating.

Stream-parseable

Events emitted as data is read — no full-document buffering required. Process data as it arrives, not after it's all loaded.

Spec-projected

Consumer-side .spec.pakt files enable projections: materialize only the fields you need, skip everything else without allocation.
